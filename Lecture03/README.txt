These code segments follow the order of Lecture 03 for CSE 20133

Follow the order of the slide set found at https://sites.google.com/a/nd.edu/morrison/courses/cse20133/lecture-notes/

The order of compilation and files to understand the design is as follows:

-------------------------------------------------
Command 1 - make nothing

Files: nothing.cpp

Slides: 5-6

nothing.cpp shows the student how code works, even when there appears to be nothing going on.

-------------------------------------------------
Command 2 - make hello

Files: hello.cpp

Slides: 7-8

This file shows an example of a classic C++ Hello, World program

-------------------------------------------------
Command 3 - make specifiers

Files: specifiers.cpp

Slides: 13-17

This file shows the students an introduction to C output specifiers
